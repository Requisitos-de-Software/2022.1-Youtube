    Ferramentas do Projeto

    1. Introdução

    Durante todo o desenvolvimento do projeto, a equipe utilizará diversas ferramentas de auxílio. Dessa forma, abaixo estão elas listadas para conhecimento geral.

    2. Ferramentas

    A equipe selecionou as seguintes ferramentas para serem usadas como auxílio durante o desenvolvimento do projeto ao longo do semestre:

    Logo Ferramenta Função
    Discord Reuniões da equipe.
    Figma Criação de protótipos e imagens (como, por exemplo, Rich Picture).
    Github Hospedagem do site e arquivos do projeto. Versionamento.
    Google Docs Anotações compartilhadas.
    Teams Gravações das apresentações.
    Visual Studio Code Edição das páginas (formato markdown) do site do projeto.
    WhatsApp Comunicação imediata da equipe. Ideal para resolver assuntos rápidos.
    Youtube Hospedagem dos vídeos das apresentações.
